# Image Assets

This folder contains image assets used by the extension.

## Files

### Nazi.png
- **Type**: PNG image
- **Usage**: Dashboard logo
- **Location**: Used in settings.html dashboard header

### book_128x128.png
- **Type**: PNG image (128x128 pixels)
- **Usage**: Extension icon (large)
- **Location**: manifest.json icons.128

### book_48x48.png
- **Type**: PNG image (48x48 pixels)
- **Usage**: Extension icon (medium)
- **Location**: manifest.json icons.48

### book_32x32.png
- **Type**: PNG image (32x32 pixels)
- **Usage**: Extension icon (small)
- **Location**: manifest.json icons.32

### book_16x16.png
- **Type**: PNG image (16x16 pixels)
- **Usage**: Extension icon (tiny)
- **Location**: manifest.json icons.16

## Icon Sizes

Chrome extensions require multiple icon sizes:
- 16x16: Toolbar
- 32x32: Windows computers
- 48x48: Extensions page
- 128x128: Chrome Web Store

## Note

Image files are binary assets and cannot be "decoded".
They are standard PNG format images.
